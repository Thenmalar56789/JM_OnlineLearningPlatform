JM Online Learning Platform is a responsive front-end website project designed to simulate an online education portal. It focuses on clean UI design, structured layout, and basic interactivity using vanilla JavaScript.

This project was built as part of an internship task to demonstrate front-end development skills using HTML, CSS, and JavaScript.

**Features**

- Clean and modern landing page layout  
- Fixed header with gradient background  
- Hamburger menu with toggle functionality  
- Search bar with suggestion list (previous searches)  
- Course cards with hover effects  
- Smooth transitions and animations  
- Organized sections:  
  - Hero / Search Section  
  - Quotes & Promotions  
  - Top IT Courses  
  - Trending Courses  
  - Recognition  
  - Partners  
  - Footer

1. Clone or download the repository.
2. Open `index.html` in any modern browser.
3. Interact with:
